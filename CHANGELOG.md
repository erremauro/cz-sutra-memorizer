# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]
### Added
- Added "Ji HŌ SAN SHI" audio and text to `sutra.json`
- Add "GYŌHATSU NENJU" translation to `sutra.json`
### Changed
- All sutra text nwo supports HTML tags
- Sutra description text has been shortned by default to 150 characters with a read more button.

## [1.2.0]
### Added
- Added the fueko dedication audio and text to `sutra.json`.
- Added `taggesa-ge.m4a` and updates its timing and translation.
- Added URL support for selected sutra and part.
### Changed
- Updated sutra title in `sutra.json` as an object
- Updated `hannya-shingyo.m4a` and related timings in `sutra.json`

## [1.1.1]
### Fixed
- Fix control styling and appearance.

## [1.1.0]
### Added
- Add the audio track for "SHIGU SEIGAN MON" and "GYŌHATSU NENJU" sutras
### Changed
- Update the names of the Sutras
- Update the styles and presentation
- Migrate the audio to m4a
- The Sutra's selector is now a filterable dropdown

## [1.0.0]
## Changed
- Change the "romaji" label to "text" label
- Add the daitokuji and bell hit symbols to the hannya shingyo text

### Fixed
- Fix Carousel navigation and audio playback
- Fix card's audio control behaviour.
